flask-sample-app
===

[Flask](http://flask.pocoo.org/)で簡単なWebアプリを作るサンプル

# Requirements
```
$ pyenv --version
pyenv 20151006

$ pyenv virtualenv --version
pyenv-virtualenv 20151006

$ node --version
v4.2.1

$ npm --version
2.14.7
```

- [pyenv](https://github.com/yyuu/pyenv)
- [pyenv-virtualenv](https://github.com/yyuu/pyenv-virtualenv)

# Setup
```
$ make setup
$ make install
```
